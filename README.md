# Next Template⚡️

⚡️A base Next starter template

## Dependencies

- next 12
- react 18
- typescript
- tailWind
- eslint
- prettier
- commitizen

## Getting Started

```shell
npm install degit -g

# yarn
npx degit ShowMeTheCodePLZ/cna-template-base myapp
cd myapp
yarn
git init
npx husky install

# npm
npx degit ShowMeTheCodePLZ/cna-template-base myapp
cd myapp
npm i
git init
npx husky install

```

### Prerequisites

- `npm` should be installed.
- `git` should be installed (recommended v2.4.11 or higher)
